# Website-Inhalte pflegen — Anleitung für das Team

**Modernes Theater Weinheim · Stand: Juli 2026**

Diese Anleitung erklärt, wie Sie Filme, Vorstellungen, Events und Saal-Informationen
auf der Website pflegen. Sie brauchen dafür keine technischen Vorkenntnisse.

---

## 1. Anmelden

1. Öffnen Sie im Browser: **https://kinoweinheim.sanity.studio**
2. Melden Sie sich mit Ihrer eingeladenen E-Mail-Adresse an (Google-Konto oder E-Mail-Link).
3. Links sehen Sie drei Bereiche: **Saal**, **Film** und **Event**.

> **Kein Zugang?** Sie müssen einmalig per E-Mail eingeladen werden — wenden Sie
> sich an die Website-Betreuung.

---

## 2. Das wichtigste Konzept: „Publish"

Alles, was Sie eintragen, ist zunächst ein **Entwurf** — auf der Website noch
unsichtbar. Erst der blaue **Publish**-Knopf unten rechts macht die Änderung
öffentlich.

- **Entwurf** = nur Sie sehen es im Studio (kleiner Punkt am Dokument zeigt
  ungespeicherte Veröffentlichung an)
- **Publish** = geht auf die Website

Nach dem Publish baut sich die Website automatisch neu — das dauert
**etwa 2–3 Minuten**. Laden Sie die Website danach neu, um Ihre Änderung zu sehen.

> **Fehler gemacht?** Oben rechts im Dokument über das „···"-Menü können Sie
> mit „Restore" eine frühere Fassung zurückholen — allerdings nur innerhalb
> weniger Tage. Im Zweifel lieber vorher fragen als löschen.

---

## 3. Einen Film anlegen

**Film** (linke Leiste) → **+** (oben) → Felder ausfüllen:

| Feld | Was eintragen |
|---|---|
| **Titel** | Filmtitel, so wie er auf der Website stehen soll |
| **URL-Kürzel** | Auf „Generate" klicken — wird automatisch aus dem Titel erzeugt |
| **Plakat** | Filmplakat hochladen (siehe Bild-Tipps unten) |
| **Länge in Minuten** | z. B. `112` |
| **FSK** | aus der Liste wählen |
| **Genre** | aus der Liste wählen (optional) |
| **Kurzbeschreibung** | 1–3 Sätze, erscheinen auf der Film-Detailseite |
| **Trailer-URL** | YouTube-Link zum Trailer (optional). Der Player erscheint automatisch auf der Detailseite |

**Die Schalter** (an/aus):

- **In 3D verfügbar** — Film läuft (auch) in 3D
- **Ist Sneak Preview** — für die Sneak-Reihe
- **Gehört zu einer Sonderreihe** — allgemeine Kennzeichnung
- **Kaffee-Tee-Kino** — Film gehört zur Kaffee-Tee-Kino-Reihe (erscheint im Filter)
- **Neu** — zeigt ein rotes „Neu"-Schild am Film. **In der Startwoche anschalten, danach wieder ausschalten!**
- **Preview** — Vorpremiere vor dem offiziellen Start
- **OmU** — Original mit Untertiteln (erscheint im Filter)

**Ganz wichtig — das Feld „Status":**

- `Entwurf` = Film erscheint **nirgends** auf der Website
- `Aktiv` = Film erscheint im Programm
- `Archiviert` = Film ist aus dem Programm genommen

Ein neuer Film muss also auf **Aktiv** gestellt UND **published** werden.

---

## 4. Vorstellungen eintragen

Im Film-Dokument, Feld **Vorstellungen** → **Add item**:

1. **Datum** wählen
2. **Uhrzeit** im Format `HH:MM` eintragen, z. B. `20:15`
3. **Saal** auswählen (Chic, Cherie, Charme oder Kinogarten)
4. **Format**: 2D oder 3D

Für jede einzelne Vorstellung ein eigenes Item anlegen — auch wenn der Film
mehrmals am selben Tag läuft. Die Website gruppiert die Termine automatisch
nach Datum.

**Vergangene Vorstellungen** müssen Sie nicht löschen — die Website blendet
sie automatisch aus. Einmal im Monat aufräumen schadet aber nicht.

---

## 5. Ein Event anlegen

**Event** → **+** → Felder ausfüllen:

| Feld | Was eintragen |
|---|---|
| **Titel** | z. B. „Kino Open-Air am Fels" |
| **URL-Kürzel** | „Generate" klicken |
| **Kategorie** | bestimmt die Farbe der Event-Karte auf der Website |
| **Kurzbeschreibung** | 1–3 Sätze für die Übersichtskarte |
| **Ausführliche Beschreibung** | beliebig langer Text für die Detail-Seite — hier können Sie Absätze, Überschriften und Listen formatieren |
| **Startdatum** | Pflicht — auch bei wiederkehrenden Reihen den nächsten Termin eintragen |
| **Enddatum** | nur bei mehrtägigen Events |
| **Wiederkehrend?** | Freitext wie „Jeden 1. Mittwoch im Monat" — wird statt des Datums angezeigt |
| **Ort** | Saal-Name oder externer Ort (z. B. „Wiesensee, Hemsbach") |
| **Bild** | optional, erscheint auf der Detail-Seite |
| **Veröffentlicht** | muss AN sein, sonst erscheint das Event nicht |

**Abgelaufene Events** verschwinden automatisch von der Startseite. Auf der
Events-Seite bleiben sie sichtbar, bis Sie sie löschen oder „Veröffentlicht"
ausschalten.

---

## 6. Säle bearbeiten

Die vier Säle (Chic, Cherie, Charme, Kinogarten) sind bereits angelegt —
Sie können Texte, Bilder und Fakten jederzeit ändern:

- **Hauptbild** — das große Foto neben der Beschreibung
- **Fakten-Kacheln** — bis zu drei kleine Zahlen-Kacheln (z. B. „201 / Plätze")
- **Ausführliche Beschreibung** — längerer Text unter den Fakten
- **Reihenfolge** — Zahl bestimmt die Sortierung auf der Säle-Seite (1 = oben)

Die Farbfelder (Farbakzent, Hintergrundfarbe, Dunkler Hintergrund) sind
Design-Einstellungen — im Zweifel nicht ändern.

---

## 6a. Preise & Öffnungszeiten ändern

Links in der Liste finden Sie den Eintrag **„Preise & Öffnungszeiten"**
(unterhalb des Trennstrichs). Das ist ein einzelnes Dokument — hier gibt
es kein „Neu anlegen", Sie bearbeiten immer denselben Eintrag:

- **Eintrittspreise** — die Preis-Kacheln auf der Kontakt-Seite.
  Empfohlen sind genau vier Einträge, damit das Raster gleichmäßig bleibt.
  „Hervorheben" gibt einer Kachel einen goldenen Rahmen (z. B. Kinotag).
- **Zuschläge & Sonderpreise** — die Liste darunter, je Zeile ein Eintrag
  (z. B. „3D +3,00 €").
- **Öffnungszeiten: Kasse / Zusatz** — die beiden Absätze unter
  „Öffnungszeiten". Der Hinweis auf das aktuelle Programm wird
  automatisch angehängt.

Wie überall gilt: Erst nach **Publish** erscheint die Änderung auf der
Website (nach wenigen Minuten).

---

## 7. Bild-Tipps

- **Filmplakate**: Hochformat, mindestens 600 Pixel breit. Das Original-
  Seitenverhältnis bleibt auf der Website erhalten.
- **Saal-/Event-Fotos**: Querformat, mindestens 1200 Pixel breit.
- Nach dem Hochladen können Sie mit dem **Hotspot-Werkzeug** (Stift-Symbol
  auf dem Bild) den wichtigsten Bildbereich markieren — er bleibt bei
  Zuschnitten immer sichtbar.
- **Urheberrecht**: Nur Bilder hochladen, für die das Kino Nutzungsrechte
  hat (eigene Fotos, Verleiher-Pressematerial).

---

## 8. Häufige Probleme

| Problem | Lösung |
|---|---|
| „Mein Film erscheint nicht auf der Website" | 1. Status auf `Aktiv`? 2. **Publish** geklickt? 3. Zwei bis drei Minuten gewartet und Seite neu geladen? |
| „Mein Event fehlt auf der Startseite" | Die Startseite zeigt nur die **nächsten 3 kommenden** Events. Liegt das Startdatum in der Vergangenheit? |
| „Ich habe etwas kaputt gemacht" | Nichts löschen, Ruhe bewahren — über „···" → „Restore" lässt sich eine frühere Fassung zurückholen. Im Zweifel Website-Betreuung anrufen. |
| „Das URL-Kürzel hat einen Tippfehler" | Korrigieren und neu publishen ist okay — aber nur solange die Seite noch nicht öffentlich verlinkt wurde. Danach lieber die Website-Betreuung fragen. |
